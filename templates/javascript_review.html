Sample JavaScript Function

Example

Function format

   <script>
   function myFunction() { # function name with potential parameters
	document.getElementById('demo').innerHTML = Date() # potential arguments/variables
	return  # return statement
	
	}
	
   myFunction() # calling the function	
   </script>

   Below is the same example mapped to a button.

Note:

  • Functions in JavaScript are ways to organize tasks that run when they are called.
  • Functions are declared in <script> tags.
  • The <script> tag defines that this document is using JavaScript elements.
  • Functions can be separated into three sections: Header, Body and Return Statement
  • The function header contains the function name and any potential parameters. Parameters pass values to a function.
  • The function body contains what you want the function to execute.
  • The function return statement contains what is sent back to the function call.

JavaScript Variables

Example

# Declaration with let
 let x = 5; # stores the value 5
 let y = 4; # stores the value 4
 let z = x + y; # stores the value 9

# Declaration with const
 const x = 5; # stores the value 5
 const y = 4; # stores the value 4
 const z = x + y; # stores the value 9

Note:

  • Variables store values for elements in JavaScript.
  • In this case, this declares values storing integers using let and const.
  • Variables declared by let can be reassigned at any time.
  • Variables declared by const are constants that cannot be changed once declared.
  • All variable declarations end with a semicolon.
