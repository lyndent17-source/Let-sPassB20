Introduction to CSS

CSS stands for Cascading Style Sheets.

CSS consists of a series of elements that tell the browser how to design and decorate the content.

A Sample CSS Document

Example


<!DOCTYPE html>
<html>
  <head>
    <style>
      div.example {
        background-color: lightblue;
      }

      h1 {
        color: white;
        text-align: center;
      }

      p {
        font-family: verdana;
        font-size: 20px;
      }
    </style>
  </head>
  <body>
    <div class="example">
      <h1>This is a white header</h1>
      <p style="text-align: center;">This is a centered paragraph.</p>
    </div>
  </body>
</html>

Note:

  • The <style> tag defines that this document is using CSS rules.
  • CSS rules generally contain a selector and a declaration block.
  • The selector is the element you want the styling to be applied to.
  • The declaration block contains all the properties you want to apply to the element.
  • Every declaration includes a CSS property and value, separated by a colon.
  • Selectors can contain multiple declarations separated by semicolons.

CSS Selectors

Example


h1 {
  color: blue;
}

.example {
  background-color: lightgray;
}

#special {
  font-size: 24px;
}

Note:

  • A selector is used to target HTML elements for styling.
  • An element selector targets all elements of a given type, such as h1 or p.
  • A class selector starts with a dot, such as .example.
  • An id selector starts with a hash symbol, such as #special.
  • Selectors help apply different styles to different parts of a webpage.

The Box Model

Example

This is a box with green borders and a blue background

Note:

  • The CSS Box Model is a box that wraps around every HTML element.
  • It consists of content, padding, border, and margin.
  • Padding is the space between the content and the border.
  • Border surrounds the padding and content.
  • Margin is the space outside the border.
