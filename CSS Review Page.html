Introduction to CSS

CSS stands for Cascading Style Sheets.

CSS consists of a series of elements, they tell the browser how to design and decorate the content.

Note:

  • The <style> tag defines that this document is using CSS elements.
  • CSS rules generally contain a selector and declaration block.
  • The selector is the element you want the styling to be applied to.
  • The declaration block contains all the properties you want to apply to the element.
  • Every declaration block includes a CSS property and value, separated by colons.
  • Selectors can contain multiple Declaration blocks separated by semi-colons.
